# Issue Templates

Guides creation and refinement of issue specifications.

## When to Apply

- Creating new issues
- Triaging raw captures
- Refining backlog items
- User asks about templates

## Templates

Located in `.docflow/templates/`:
- `feature.md` - New functionality
- `bug.md` - Defect reports
- `chore.md` - Maintenance work
- `idea.md` - Future exploration
- `quick-capture.md` - Fast capture

## Usage

1. Read appropriate template
2. Follow `<!-- AGENT INSTRUCTIONS -->` comments
3. Apply structure to issue
4. Remove instruction comments from final issue
