# DocFlow Update (System)

Check for and apply updates to DocFlow rules.

## Steps

1. **Check Version** - Compare `.docflow/version` to source
2. **Show Changelog** - What's new (if available)
3. **Sync Rules** - Update if user approves
4. **Confirm** update complete

## Source Repository

Updates come from: `github.com/strideUX/docflow-template`

## What Gets Updated

- `.docflow/rules/` - Workflow rules
- `.docflow/templates/` - Issue templates
- `.cursor/rules/` - Cursor rule structure
- `.cursor/commands/` - Command files

## Full Rules

See `.docflow/rules/core.md` for version info.
