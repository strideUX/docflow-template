# Attach (Implementation Agent)

Attach files to a Linear issue.

## Methods

1. **GitHub Link** - For committed files
2. **Direct Upload** - For local files

## Steps

1. **Determine Method** - GitHub link or upload
2. **For Committed Files** - Create GitHub permalink
3. **For Local Files** - Upload to Linear storage
4. **Create Attachment** - Link to issue
5. **Confirm** with attachment link

## Natural Language Triggers

- "attach [file]" / "link the doc" / "upload notes"
